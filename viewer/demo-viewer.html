SVGTreeViewer Demo

The examples on this page showcase the basic functionality of the SVGTreeViewer widget. The code fragments below use jQuery. It is not required by SVGTree, but used to simplify the code of examples.

Customizable Actions

	
		

		
		
You can disable undo, settings and / or Newick notation buttons by specifying appropriate boolean options (undo, settings, newick).

Footer Removal

	
		

		
		
Alternatively, the footer containing node-specific actions can be removed altogether using the footer option.
		The footer is removed automatically if tree nodes cannot be selected.

Widget Size

	
		
You can specify the size of a widget using width and / or height styles (e.g., using <div ... style="width: 500px; height: 500px;">).
		Note that for the optimal layout, the width and the height of the widget should be no less than 300px.

For large trees, there is a useful Locate button which centers the displayed area of the tree on the selected node.
